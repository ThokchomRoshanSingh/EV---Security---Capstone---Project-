# EV---Security---Capstone---Project-
Capstone Project - EV Security Risk Analysis and Simulation 
Project Overview

This capstone project focuses on identifying potential cybersecurity threats in Electric Vehicle (EV) charging infrastructure. As EV adoption increases, charging stations become part of critical infrastructure and may become targets for cyberattacks.

The goal of this project is to simulate EV charging behavior, generate a dataset containing normal and malicious activities, and analyze potential security risks using an Intrusion Detection System (IDS) approach.

Objectives

1. Analyze cybersecurity risks in EV charging infrastructure.
2. Simulate EV charging sessions and potential attack scenarios.
3. Generate a synthetic dataset for IDS training and testing.
4. Detect abnormal behaviors such as denial-of-service attacks and spoofing attempts.
5. Evaluate system performance for detecting cyber threats.

Attack Scenarios Simulated

The dataset includes several simulated attack behaviors commonly discussed in EV charging security research:
1. Denial of Service (DoS) – Overloading the system with excessive requests.
2. False Data Injection – Manipulating charging parameters or telemetry data.
3. Authentication / Spoofing Attempts – Unauthorized access attempts to charging services.
